Sweep

Know what you have. Find what you need. Stop losing files in the pile.
Phase 1 Shipped Python 3.10+ SQLite Zero Dependencies

The Problem

You hoard data to avoid losing it, then lose data because you have too much. 2, 3, 4 copies of the same thing scattered across drives and network shares. Millions of files across dozens of folders. No way to know what's where without manually clicking through File Explorer for hours.

The Solution

Sweep crawls your drives and network shares, catalogs every file and folder, and stores the results in a searchable SQLite database. Run it once, walk away, come back to a complete inventory. Kill it anytime — it picks up where it left off.

Key Features

  • Set it and forget it — point it at paths, run one command, walk away for hours or days
  • Unkillable — blue screen, power outage, Ctrl+C. Doesn't matter. It resumes exactly where it stopped.
  • Network-safe — configurable throttle so you don't hammer shared drives or get flagged by IT
  • Find anything — query by file extension, path, or size from the command line
  • Zero dependencies — Python stdlib only. No pip, no venv, no admin rights. Copy and run.
  • Two environments, one tool — home mode (full speed, dedup) and work mode (throttled, catalog-only)

What It's NOT

  • Not a file sync tool (it doesn't move or copy files)
  • Not a backup tool (it catalogs, it doesn't protect)
  • Not a search engine (no full-text content search — metadata only)
  • Not a dedup tool (it finds duplicates — it doesn't delete them)
  • Phase 1 is CLI only — no web UI yet
